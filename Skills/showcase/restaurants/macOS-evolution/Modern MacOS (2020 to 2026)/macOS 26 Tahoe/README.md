## 🛠️ THE ARCHITECTURE (GEMINI BLUEPRINT)
### **Core System Logic**
1. **System Lifecycle**: Seamless flow from `BootSequence` (432Hz sine chime) through `SetupAssistant` to the `Desktop`.
2. **Virtual File System (VFS)**: A persistent, `localStorage`-backed drive managing unique node IDs, trash recursion, and metadata.
3. **Genie Effect Windowing**: Custom cubic-bezier transition paths simulating windows "sucking" into the Dock.
4. **App-Aware Menu Bar**: A dynamic subscriber that re-scripts its menus (File, Edit, Go) based on the `activeApp` focus.

### **Technical Stack**
- **Framework**: React 18 + Vite (TypeScript)
- **Styling**: Tailwind CSS (Custom glassmorphism utilities)
- **Animation**: Framer Motion (Transitions, Physics, Magnification)
- **Icons**: HugeIcons + Lucide React
- **Persistence**: `tahoe_v3_state` & `tahoe_v3_fs` in LocalStorage.

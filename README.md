# macOS Tahoe OS v4 (Implementation)
### **The high-fidelity React 18 core for the macOS Tahoe Ecosystem.**

This directory contains the production-ready source code for the macOS Tahoe simulation.

---

## ⚡ Quick Start
```bash
npm install
npm run dev
```

## 🏗️ Architecture
- **Framework**: React 18 + Vite (TypeScript)
- **State**: Centralized `SystemContext` + Persistent `FileSystemContext`.
- **UI Engine**: Tailwind CSS + Framer Motion (120fps Liquid Glass physics).

## 🚀 Key Features
- **Hardware Telemetry**: Real-time Battery/Memory/Storage hooks.
- **Apple Intelligence**: Writing Tools & Genmoji.
- **Dynamic Desktop**: 4-stage 24h wallpaper cycle.
